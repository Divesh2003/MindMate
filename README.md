# MindMate

Welcome to MindMate, a web app designed to analyze and address psychological and mental health concerns. Our goal is to provide users with easy access to resources for mental well-being. MindMate helps individuals identify issues early and take necessary steps to improve their mental health. Our aim is to provide solutions without always requiring professional help. We've seen our approach effectively improve mental health conditions. Users can track their habits and behaviors, and in serious cases, book appointments with psychologists for consultations and timely assistance.

## Install, Configure & Run

Install Anaconda from Official Website

```bash
  https://www.anaconda.com/download/
```

Download Repo from Github

```bash
  git clone https://github.com/Divesh2003/MindMate.git
```

Create a Virtual Environment using Conda

```bash
  conda create --name myenv python=3.7
```

& Follow These Steps

```bash
  cd MindMate
```

```bash
  conda activate myenv
```

```bash
  pip install -r requirements.txt
```

```bash
  flask run
```
Or
```bash
  conda run -n myenv flask run
```

Open Browser & Run
```bash
  http://127.0.0.1:5000/
```
